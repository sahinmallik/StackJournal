# About This Project

Welcome to our documentation platform 🚀

This website is built using **Next.js** for the frontend and modern tooling to provide fast, structured, and developer-friendly documentation.

---

## 🎯 Our Mission

Our goal is simple:

- Provide clean and structured documentation
- Make content easy to navigate
- Support Markdown & MDX for flexibility
- Deliver blazing fast performance

---

## 🛠 Tech Stack

- **Next.js** – React framework for production
- **MDX** – Markdown with React components
- **TypeScript** – Type-safe development
- **Tailwind CSS** – Utility-first styling

---

## ✨ Why MDX?

MDX allows us to:

- Write Markdown
- Embed React components
- Create interactive documentation
- Maintain structured content

Example:

```jsx copy
export function Highlight({ children }) {
  return <span className="text-blue-500 font-semibold">{children}</span>;
}
```
